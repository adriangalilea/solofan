# Release Notes — v1.6.2 (2026-05-26)

## Highlights

- **Settings polish** — Flat window background (no animated gradient).
- **Floating glass sidebar** — Wider 268pt Liquid Glass navigation panel with larger tabs and clearer selection.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/mohamadlounnas/ffan/main/scripts/install.sh | bash
```

Or download `solofan-v1.6.2-macos.dmg` from [GitHub Releases](https://github.com/mohamadlounnas/ffan/releases/latest).

## Requirements

- macOS **26.1** or later
- Apple Silicon or Intel
