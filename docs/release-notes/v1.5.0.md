# Release Notes — v1.5.0 (2026-05-24)

## SoloFan rebrand

The app is now **SoloFan** (`SoloFan.app`). Download assets use the `solofan-v1.5.0-macos.{zip,dmg}` naming pattern.

## Highlights

- Liquid Glass settings UI (macOS 26 native `.glassEffect`)
- Per-fan SMC RPM limits and optional per-fan manual sliders
- Menu bar right-click: Open Settings, Hide Icon, Close App
- Hidden icon mode with settings on relaunch

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/mohamadlounnas/ffan/main/scripts/install.sh | bash
```

Requires **macOS 26.1+**.
