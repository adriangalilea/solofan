# Release Notes — v1.6.1 (2026-05-24)

## Highlights

- **Settings redesign** — Native macOS `NavigationSplitView` with system sidebar glass, grouped Form content (no glass-on-glass), aligned with the [Liquid Glass Reference](https://github.com/conorluddy/LiquidGlassReference).

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/mohamadlounnas/ffan/main/scripts/install.sh | bash
```

Or download `solofan-v1.6.1-macos.dmg` from [GitHub Releases](https://github.com/mohamadlounnas/ffan/releases/latest).

## Requirements

- macOS **26.1** or later
- Apple Silicon or Intel
