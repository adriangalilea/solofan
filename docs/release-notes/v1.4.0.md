# Release Notes — v1.4.0 (2026-05-10)

## Highlights

- **Correct fan limits**: Maximum and minimum RPM come from SMC **per fan**, fixing incorrect global **6500 RPM** behavior on Apple Silicon (e.g. M1 Pro left/right fans with different ceilings).
- **Per-fan manual targets**: Toggle separate sliders for each fan when more than one fan is present (manual mode).
- **Consistent metrics**: Gauge and menu bar fan **%** use each fan’s actual min–max span.

## Compatibility

- **macOS 26.1 or later** — matches `MACOSX_DEPLOYMENT_TARGET` in the Xcode project. Update docs previously implied older macOS; **current release binaries require 26.1+**.

## Installation

Use [GitHub Releases](https://github.com/mohamadlounnas/ffan/releases/latest) (ZIP/DMG from CI when tagged), or:

```bash
curl -fsSL https://raw.githubusercontent.com/mohamadlounnas/ffan/main/scripts/install.sh | bash
```
