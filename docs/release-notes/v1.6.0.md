# Release Notes — v1.6.0 (2026-05-24)

## Highlights

- **App icon fix** — SoloFanIcon generated from `docs/assets/logo.png` (all macOS sizes).
- **Widget dashboard** — Customize the menu bar popover: add/remove widgets, drag to reorder, 1–2 columns per row, **Reset** to defaults.
- **Desktop Macs** — CPU + GPU cards when no battery; status bar power mode falls back to fan % or temperature.
- **Settings** — Liquid Glass aligned with Apple HIG: glass on navigation only, plain grouped content.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/mohamadlounnas/ffan/main/scripts/install.sh | bash
```

Or download `solofan-v1.6.0-macos.dmg` from [GitHub Releases](https://github.com/mohamadlounnas/ffan/releases/latest).

## Requirements

- macOS **26.1** or later
- Apple Silicon or Intel
